CSS – Cascading StyleSheets

CSS – allows control on how HTML elements will look on a web browser.

A selector selects the HTML element that will be styled. Example: h1 {color: red;}, color is the property, and red is the value. (Declarations)

{ } – are curly braces that define the declarations for the selector.

W3C, WHATWG, & Khronos- are some of the standard web technologies that define precise specifications on how they are to behave.

A link is an element that tells the browser there’s a stylesheet, using rel attribute, and the location of the stylesheet as the value of href attribute.

To remove list bullets on a web page, use list-style-type: none; in CSS stylesheet.

The class attribute selects all elements to look the same. Example: li class = “special” defined in HTML; .special {color: orange;} defined in CSS.

Three kinds of stylesheets

  • External – files with extension using .css. This is used in a HTML webpage with a link element.
  • Internal – defined inside HTML webpage with a unique style, defined inside style element.
  • Inline – use style attributes and used very rarely. This style will override external and internal styles.

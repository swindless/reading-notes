How the computer works – The Input and Output

The computer is considered a thinking machine that performs multiple tasks:

  • Input – what the user tells the computer what to do.
  • Storage - handle the information stored in memory.
  • Process – a series of commands that accepts the input in the computer and then sends it back to the computer for output – this is called algorithm.
  • Output - can display text, photos, videos, or interactive games. When it comes to the internet, output can pass through and become someone’s input and vise versa.

Learning JavaScript

JavaScript is known for a scripting and interpreted language. It involves compilation during execution of a program at run time rather than before execution, which is known as (just in time (JIT). It is a compiled program language with first class functions.

A function can pass an argument to other functions, can be returned by another function, and can be assigned as a value to a variable. It is a block of code designed to perform a particular task or when an event occurs – like when a user clicks on a button.

The script element is used to embed executable code or data. It can be placed in the body or head section of the HTML webpage or in both.

The external JavaScript is a file with a filename extension .js that is also created in the same HTML file.

The external JavaScript can be referenced in 3 ways:

  • With the URL
  • With the file path (/js/)
  • Without the path

The confirm() method is used to display a dialog with an optional message and two buttons, OK and Cancel. It returns true if the user clicks “OK”, and false otherwise. It prevents the user from accessing other parts of the page until the box is closed.

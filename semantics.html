Learning about Semantics

Semantic elements are elements that refer to a meaning of piece of code. It describes its meaning to both browser and the developers. It defines different parts of a web page.

Some examples of semantics are:

  • nav -provide navigation links, either within the current document or to other documents. Navigation sections are menus, tables of contents, and indexes.
  • header - represents introductory content (may contain a logo, a search form, and other elements)
  • footer - contains footer content (copyright information, sitemap, & contact information)
  • article - blog posts, user comments, newspaper articles
  • section - grouping of content (chapters, introduction, new items)
  • figure - specifies content like illustrations, diagrams, photos
  • figcaption defines a caption
  • aside - sidebar information about the related contents of the web page
  • details - an interactive widget that a user can view and hide details. By default, the widget is closed, when opened it by the user, it expands and displays the content.

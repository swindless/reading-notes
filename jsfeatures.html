What is Javascript?

Javascript builds the application for interactive web pages. Javascript is the behavior or how it should act on a webpage. HTML is the structure or how it will look on a webpage.

Where do we use Javascript?

  • web and mobile apps
  • real time networking apps
  • command line tools
  • video games

Javascript can display data in different ways:

  • Writing into an HTML element, innerHTML
  • Writing into an HTML output, document.write( )

  • Writing in the browser console, console.log ( )

A javascript function is executed when something invokes it and is followed by ( ). Parameters are the names inside of the ( ). The arguments are values received by the function when invoked or called.
